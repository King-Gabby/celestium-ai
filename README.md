# CelesTium AI

## Building Computer Vision Infrastructure for Africa

CelesTium AI is an African computer vision company developing intelligent systems for healthcare, transportation, and critical real-world operations.

### Focus Areas

#### Healthcare AI

* Malaria Detection System
* Medical Imaging Intelligence
* Clinical Decision Support

#### Transportation Intelligence

* Automatic Number Plate Recognition (ANPR)
* Traffic Monitoring
* Vehicle Analytics

#### AI Infrastructure

* Computer Vision Pipelines
* Dataset Engineering
* Model Deployment Systems

---

## Current Projects

### Malaria Detection System

An AI-powered malaria parasite detection platform built using YOLOv8 and microscopy image analysis.

Features:

* Parasite detection and localization
* Stage classification
* Parasitemia estimation
* Clinical reporting interface

### ANPR Platform

Computer vision system for automatic vehicle number plate recognition and transportation intelligence.
